Section 5: Date-Converter utility

I have released a generic date-conversion utility on Maven Central Repository to convert any date-object of any type to any other date object type. And when it comes to converting date-strings to a date-type object, the date-patterns are automatically deduced and there is no need to provide the date-pattern explicitly. This utility is used in the OTC generated source code whenever date fields are to be converted. This utility can be used in any of your software projects in Java and it is made available on Maven central.
